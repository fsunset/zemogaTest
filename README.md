# zemogaTest
Frontend Test for Zemoga. Contains the test described here --> https://github.com/zemoga/ui-test

### Techs used, among other:

* NPM
* React (react-create-app)
* React-router-dom
* SASS
* React-bootstrap
* React-fontawesome


### To run it locally
* Clone repo
* Run *npm install* to install dependencies
* Run *npm start*
